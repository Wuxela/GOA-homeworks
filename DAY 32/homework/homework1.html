CSS კონსპექტი

როგორ მუშაობს CSS

CSS (Cascading Style Sheets) არის სტილის ფაილი, რომელიც გამოიყენება HTML ელემენტების ვიზუალური წარმოდგენის განსაზღვრისთვის. CSS საშუალებას გვაძლევს ვაკონტროლოთ ფერები, შრიფტები, განლაგება და სხვა ვიზუალური ასპექტები.

სტილების მემკვიდერობა

CSS-ში სტილები მემკვიდრეობით გადაეცემა მშობელ ელემენტიდან შვილ ელემენტზე. მაგალითად, თუ მშობელ ელემენტს აქვს განსაზღვრული ფერი, ეს ფერი ავტომატურად გადაეცემა ყველა შვილ ელემენტს, თუ მათთვის არ არის განსაზღვრული სხვა ფერი.

ელემენტებზე წვდომა და არჩევა

CSS-ში ელემენტებზე წვდომა ხდება სელექტორების საშუალებით. არსებობს სხვადასხვა ტიპის სელექტორები, როგორიცაა ელემენტის სელექტორები (მაგალითად, h1, p), კლასის სელექტორები (.classname), ID სელექტორები (#idname) და კომბინირებული სელექტორები (მაგალითად, div p).

Specificity ალგორითმი

Specificity ალგორითმი განსაზღვრავს, რომელი CSS წესი იქნება გამოყენებული, როდესაც რამდენიმე წესი მიმართულია ერთსა და იმავე ელემენტზე. Specificity ითვლება შემდეგი წესებით:

  • Inline სტილები (style attribute) - ყველაზე მაღალი Specificity
  • ID სელექტორები (#idname)
  • კლასის სელექტორები (.classname), ატრიბუტის სელექტორები და პსევდო-კლასები (:hover)
  • ელემენტის სელექტორები (h1, p) და პსევდო-ელემენტები (::before)

თუ ორი წესი აქვს ერთსა და იმავე Specificity-ს, მაშინ ბოლო განსაზღვრული წესი იქნება გამოყენებული.

CSS საუკეთესო პრაქტიკა და სტანდარტების დაცვა

CSS-ის დაწერისას საუკეთესო პრაქტიკისა და სტანდარტების დაცვა მნიშვნელოვანია კოდის ხარისხისა და შენახვისთვის. აქ არის რამდენიმე რჩევა:

  • გამოიყენეთ სელექტორები რაც შეიძლება სპეციფიკურები, მაგრამ არა ზედმეტად რთულები.
  • გამოიყენეთ კლასები და ID-ები ლოგიკური და აღწერითი სახელებით.
  • გაითვალისწინეთ კოდის ორგანიზაცია და სტრუქტურა, რათა CSS ფაილი იყოს ადვილად წაკითხული და შენახული.
  • გამოიყენეთ კომენტარები კოდის განმარტებისთვის.
  • გამოიყენეთ CSS პრეპროცესორები (მაგალითად, SASS ან LESS) უფრო მოწინავე ფუნქციების გამოყენებისთვის.
  • გაითვალისწინეთ ბრაუზერის თავსებადობა და გამოიყენეთ ვენდორული პრეფიქსები საჭიროების შემთხვევაში.
